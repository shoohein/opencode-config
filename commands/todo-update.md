---
description: git 履歴やファイル変更から TODO.md の状態を更新
---

`TODO.md` を読み、現在の TODO 一覧を把握する。

次に:

1. `git log --oneline -30` で最近のコミットを確認
2. `git diff --name-only HEAD~5..HEAD` で最近変更されたファイルを確認
3. コミットメッセージや変更ファイルと TODO アイテムをクロスリファレンス
4. 完了したと推定されるアイテムがあれば、ユーザーに Done への移動を提案（関連コミットハッシュを添える）
5. 他に追加すべきアイテムがないか確認する
